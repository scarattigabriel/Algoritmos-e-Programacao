## 📂 Estrutura do Repositório

O conteúdo está organizado cronologicamente seguindo a evolução da ementa da disciplina:

* `01-Introducao/` - Primeiros passos com a linguagem e conceitos básicos.
* `02-Comandos Condicionais/` - Estruturas de decisão (`if`, `elif`, `else`).
* `03-Comandos de Repeticao/` - Laços de repetição (`while` e `for`).
* `04-Controle e Interrupcao/` - Uso de `break`, `continue` e controle de fluxo.
* `05-Armazenamento de Dados Sequenciais/` - Listas, tuplas e manipulação de sequências.
* `06-Funcoes/` - Modularização de código, escopo de variáveis e parâmetros.
* `07-Estruturas Multidimensionais/` - Matrizes e listas de listas.
* `08-Estrutura para Dados Relacionados/` - Dicionários e mapeamentos.
* `09-Outras Estruturas/` - Conjuntos (sets) e estruturas complementares.
* `10-Recursividade Excecoes e Arquivos/` - Funções recursivas, tratamento de erros (`try/except`) e persistência de dados em arquivos.
* `11-Revisao/` - Exercícios de fixação e preparação para as avaliações.

## 🛠️ Como Clonar e Executar

Caso queira baixar e testar os códigos localmente:

1. Clone este repositório:

```bash
git clone https://github.com/scarattigabriel/Algoritmos-e-Programacao.git
```

2. Acesse o diretório do conteúdo desejado:

```bash
cd "03-Comandos de Repeticao"
```

3. Execute o arquivo Python(certifique-se de ter o Python instalado):

```bash
python nome_do_arquivo.py
```
